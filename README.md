# AI Chatbot Assistant (Python)

##  Description
This is a simple AI chatbot built using Python. It responds to user queries like greetings, time, and opening websites.

##  Features
- Responds to greetings
- Shows current time
- Opens YouTube and Google
- Handles user input using conditions

##  Technologies Used
- Python
- datetime module
- webbrowser module

##  How to Run
1. Install Python
2. Run the file:
   python chatbot.py

##  Concepts Used
- Functions
- Loops
- Conditional statements

## Example Output
You: hello
AI: Hi!

You: time
AI: Current time is 10:30

You: open youtube
AI: Opening YouTube...
